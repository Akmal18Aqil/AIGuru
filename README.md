# AI Guru Assistant (SiGURU)

Aplikasi desktop AI untuk membantu guru membuat RPP (Kurikulum Merdeka) dan variasi soal secara instan.

## Struktur Project
- `/src`: Source code utama (LangGraph agents)
- `/ui`: Interface pengguna (Streamlit)
- `/docs`: Dokumentasi

## Setup
1. `pip install -r requirements.txt`
2. Setup `.env` dengan `GOOGLE_API_KEY`.
3. Jalankan dengan `streamlit run ui/app.py`.
